Object-Oriented Programming

Sprint 2: Patterns, Principles, and Java Programming


Welcome to sprint 2. We covered a lot of material in sprint 1. However, we didn't get a chance to write much code. This sprint get your Java compilers out. We are going to start practicing some of the concepts we learned. Note that sprint 2 and all subsequent sprints are approximately twice as long as sprint 1 and are expected to take approximately 20 hours of focused effort to complete.

By the end of sprint 2 we will be able to:

  • Identify, define, and discuss OOP Principles
  • Create, modify, compile, and execute OOP based Java applications
  • Manage and submit source code utilizing the GitHub and the Git command line client
  • Utilize JSON to represent program data

Below is the list of activities that we will need to complete this sprint. Once again you are welcome to pursue the activities in any order you prefer. However, they are listed in the order I think will be the most beneficial for you to work through the topics. The time commitment for each line item is generally 20 to 40 minutes unless otherwise noted.

  1. Review Programming Assignment 2, Quiz 2, and Discussion Board 2
  2. Analyze and commit to adopting the “How to Be a Success Programmer” suggestions that you feel are the most important [link]
  3. Sign up for Safari Books Online which is available to you free through Lewis University [link]
  4. Within Safari Books Online execute searches for Object-Oriented Programming, OOP, Git, GitHub, and Java and review the results of each search
  5. Review the answers to Quiz 1 as many of the topics in Quiz 1 will come up in future so it is important that you understand the concepts question
  6. Complete the Java "Shapes Step 1" tutorial. As will all tutorials, be sure to follow along and complete the steps yourself, make the resulting application uniquely your own, and to be ready to submit your personalized version of the application as part of an assignment [video]
  7. Complete the Java “Shapes Step 2” tutorial [video]
  8. Complete your initial Discussion Board (DB) post by the middle of the sprint
  9. Complete “Object-Oriented Programming Principles” [video] [slides]
  10. Read and Understand the article “What is JSON? JavaScript Object Notation explained” [link]
  11. Read and Understand “JSON Introduction” from W3C [link]
  12. If you do not already have access to a current version of Microsoft Office, install the current Microsoft PowerPoint (PowerPoint Mobile) and Word (Word Mobile) viewers [video] [link]

Once again, be sure to take some time and reward yourself for your efforts. Get a coffee and spend some time reflecting on what you've accomplished. It is important to take a little time to enjoy the moment.
