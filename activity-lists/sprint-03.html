Sprint 3: Graphical User Interfaces and FaceDraw


Welcome to Sprint 3. We have covered a lot in the first two sprints. Much of it was preparing us for our first substantial programming assignment. I believe we are ready to apply what we have learned to our FaceDraw project. Be sure to get an early start on the project and stay focused. You will note that there are fewer items on our activities list this week. That is because you will need to focus your time and energy on FaceDraw.

By the end of this sprint we will be able to:

  • Develop a moderately sophisticated graphical Java application (FaceDraw)
  • Comfortably utilize our Java development environment
  • Utilize source code management and Git/GitHub
  • Effectively compile, debug, submit, update, and manage Java application source code

As you would expect, our activities list for this sprint (below) is laser focused on helping us achieve the above Learning Objectives. Once again I have attempted to list this items in the order that I think will be most beneficial for you to complete them.

  1. Review and start working on Programming Assignment 3 - FaceDraw and Quiz 3
  2. Complete "Graphical User Interfaces (GUI) in Java" lesson [video]
  3. Review the answers from Quiz 2
  4. Take one last look at "How to Be a Success Programmer" suggestions and intentionally decide if there is something you are going to change now that the programming assignments are becoming more challenging. [link]
  5. Complete the FaceDrawLite tutorial [video] ~2 hours
  6. Complete your initial Discussion Board (DB) post by the middle of the sprint
  7. FaceDraw, FaceDraw, FaceDraw ~8 to 12 hours

Take a few minutes to reflect on how much you have learned by completing the FaceDraw project. It's likely a significant step up from anything you have programmed in the past. Well done.
