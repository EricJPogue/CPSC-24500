Sprint 8! Need I say more.

With one week left, let's finish strong. I challenge you to make your final project something that makes you proud. Not because it will get you a good grade (which it likely will), but because you find it useful or elegant or beautiful.

By the end of this sprint we will be able to:

  • Independently design, develop, and enhance intermediate level graphical Java applications using Object-Oriented Programming Concepts, Patterns, and Principles
  • Understand Application Architecture and its relationship to Object-Oriented Programming
  • Understand how relational database do (and don't) support Object-Oriented Programming
